# 👥 Contributors

Папка для PR в рамках ачивки **Pull Shark 🦈**.

## Как добавить свой файл

1. [Сделай форк репо](../../../fork)
2. Создай файл `ВАШ_USERNAME.md` в этой папке
3. Открой PR — бот автоматически смёрджит его

## Правила

- Имя файла должно совпадать с твоим GitHub username: `username.md` или `username-2.md`
- Содержимое — любое (можно оставить одну строку), не больше 10 КБ
- Файлы вне этой папки — не принимаются
- Удаление чужих файлов — не принимается

## Шаблон файла

Содержимое не важно — напиши что угодно, хотя бы одну строку.
